# CreditWise Loan Prediction System

## Overview
CreditWise is a machine learning project that predicts loan approval based on applicant information. The project includes an end-to-end workflow from data preprocessing to model evaluation using classification algorithms.

## Features
- Data preprocessing and missing value handling
- Exploratory Data Analysis (EDA)
- Feature encoding and feature scaling
- Feature engineering
- Model training and evaluation
- Comparison of multiple classification models

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Machine Learning Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Encoding
- Feature Scaling
- Model Training
- Model Evaluation

## Dataset
Loan Prediction Dataset
